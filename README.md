# ReAcT-like Schedule

A React-based schedule viewer inspired by the Re:AcT talent agency's live stream schedule page.

## Overview

This project is a front-end practice build using React.
The goal is to replicate the core layout and UI of a real-world schedule listing page,
focusing on component structure, responsive design, and filter interactions.

> ⚠️ This project is currently in early development. Features and structure are subject to change.

## Tech Stack

- React (Create React App)

## Current Status
